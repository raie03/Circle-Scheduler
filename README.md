# データベース設計

## サークルテーブル

- サークル id
- イベント
- 全メンバー数

## イベントテーブル

- イベント id
- イベント名
- 出演メンバー数
- 日程調整の範囲
- 演目

## メンバーテーブル

- メンバー id
- 名前
- イベント
- 出演する演目
- 日程調整の回答
- 練習日

## 演目テーブル

- 演目 id
- 演目名
- 出演メンバー
- 練習日

## スケジュールテーブル

- 日付 id
- 日付
- 演目名
- メンバー
- 被り

## 被りテーブル

- 被り id
- 被った日時
- 被ったジャンル
- 被ったメンバー
